2024-4-15 周一

TypeScript是是什么？

1、以js为基础构建的语言，一个js的超集；

2、可以在任何支持js的平台中执行，TypeScript扩展了js并添加了类型；

3、TS不能被js解析器直接执行，TS可以直接编译JS

TypeScript增加了什么？

类型、支持ES的新特性、添加ES不具备的新特性、丰富的配置选项、强大的开发工具
